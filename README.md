Sticky Board

Sticky Board is a productivity application that combines sticky notes and task management in a single workspace. It allows users to quickly capture ideas, organize notes, and manage tasks through a Kanban-style board.

Features

• Create, edit, and delete sticky notes  
• Pin important notes  
• Search and sort notes  
• Color-coded notes for better organization  
• Create and manage tasks  
• Drag and drop tasks between columns  
• Task stages: To Do, Ongoing, Completed  
• Priority levels: Low, Medium, High, Urgent  
• Due date tracking and overdue highlighting  
• Secure user authentication with Supabase  
• Responsive and user-friendly interface  

Tech Stack

• React  
• TypeScript  
• Vite  
• Tailwind CSS  
• Supabase  
• PostgreSQL  
• TanStack Router  
• Radix UI  

Installation

Clone the repository:

```bash
git clone https://github.com/your-username/sticky-board.git
cd sticky-board
```

Install dependencies:

```bash
npm install
```

Create a `.env` file in the root directory:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

Run the development server:

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:5173
```

Build for Production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
